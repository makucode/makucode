## Startup

Dev Server:

Running on [http://localhost:3000](http://localhost:3000)

```bash
npm run dev
# or
yarn dev
```

Build:

```bash
npm run build
# or
yarn build
```

Run Build:

```bash
npm run start
# or
yarn start
```
